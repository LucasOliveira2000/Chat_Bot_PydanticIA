# Chat Bot IA
 Esse é um projeto de estudos para criar um chatbot utilizando inteligência artificial. 
 O objetivo é desenvolver um assistente virtual capaz de responder perguntas e interagir com os usuários de forma natural.

## Tecnologias Utilizadas
- Python
- Pydantic IA
- FastAPI
- Google GenAI
- PostgreSQL ( Ainda vou colocar, mas é a ideia, utilizando o vetor do pgvector para armazenar os embeddings)
- SQLAlchemy ( Ainda vou colocar, mas é a ideia, utilizando o SQLAlchemy para gerenciar o banco de dados)
